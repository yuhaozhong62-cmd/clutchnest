# Cypher × Ascent 研究记录

研究日期：2026-07-13

## 当前版本判断

- Riot Patch 13.00 将 Cypher 陷阱线启动时间从 0.9 秒缩短至 0.7 秒；公开内容仅据此确认当前技能机制，不把它误写成特定像素点验证。
- Riot Patch 9.08 记录了位移 / 召唤物可穿过 Ascent Window 并破坏相关旧陷阱的改动，因此旧 Mid Window 陷阱不进入公开战术。
- 近期 Ascent 社区战术片段仍出现 A Main / B Main 摄像头、烟笼和入口防守思路。这些资料只作为研究方向，精确落点必须由 HAO 在当前客户端复测。
- 公开的 12 条战术都以职责、区域、执行顺序和信息响应为核心；地图坐标是区域标记，不是像素级教程。

## 使用来源

- Riot Games Cypher 英雄页：技能定义。
- Riot Games Patch 13.00：当前技能版本变化。
- Riot Games Patch 9.08：排除旧 Mid Window 陷阱。
- Riot Games Maps：Ascent 地图结构与官方俯视素材。
- Tracker Network 近期 Ascent 指南片段与英雄数据：当前社区使用方向的辅助证据。

## 发布规则

- `published`：用途、区域和信息响应可解释，且当前机制未发现冲突。
- `needsVerification`：依赖精确像素、单向视野或特定附着面的内容，不在前台显示。
- 所有缺少 HAO 实机截图的公开条目都显示“待补充图片”，并在版本区说明需要实机复核的部分。
